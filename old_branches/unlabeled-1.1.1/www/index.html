      Use commands like this: 

          cvs -d :pserver:anonymous@cvs.omniorb.sourceforge.net:/cvsroot/omniorb login
    cvs -d :pserver:anonymous@cvs.omniorb.sourceforge.net:/cvsroot/omniorb co -r omni4_0_develop omni
