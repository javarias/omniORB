QoS Policies

The QoS example demonstrates setting various Quality of Service's policies which affect the behavior of VisiBroker clients. It illustrates creating policies, setting policy values, and setting the policy within the orb.

From this example, you will learn how to:
  • To create policies and set values within the policies
  • To set the policies within the orb and the effect this can have on the orb
