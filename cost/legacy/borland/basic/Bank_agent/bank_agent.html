Bank Agent

The Bank Agent has a simple AccountManager interface to open an Account and to query the balance in that account. It illustrates the use of osagent by server to publish the IOR and by client to obtain the IOR.

From this example, you will learn how to:
  • To use the right POA policy so Server registers the object IOR with osagent
  • To use the bind method in client to obtain the IOR
