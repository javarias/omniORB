Point Example

The Point example is a  simple illustration of value types, also known as Objects-By-Value (OBV).  This example illustrates how to pass an object by value, rather than by reference.  A value type provides semantics that bridge between CORBA structs and CORBA interfaces.

From this example, you will learn how to:
  • declare both regular and boxed value types in idl.
  • how to implement concrete (stateful) value types.
  • how to define factories for value types.
  • how to define and implement a value type with single inheritance.
