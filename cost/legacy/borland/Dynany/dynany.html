dynany

The DynAny example illustrates some basic facilities associated with Dynamic Any's.

From this example, you will learn how to:
  • To use the right POA policy so Server registers the object IOR with osagent
  • To use the bind method in client to obtain the IOR
  • Create and initialize a dynamic any type
  • How to create an any type from a dynamic any type.
  • ow to traverse and extract the primitive constituents of the data contained within a dynamic any type.
