BOA Bank

Visibroker 4.0 implements Portable Object Adapter (POA) functionality. However there are a number of Visibroker applications out there which are based on Basic Object Adapter (BOA). In order to support those applications, BOA is still supported in Visibroker 4.0.

This example illustrates a BOA based Bank. The code is almost identical to the BOA based bank in Visibroker 3.x. The Bank example has a simple AccountManager interface to open an Account and to query the balance in that account.


From this example, you will learn how to:
  • Use the right option in idl compiler to generate code for BOA.
  • Code modification if any to make your 3.x BOA based code to work on 4.0.
