Bank Portable

The Bank Portable has a simple AccountManager interface to open an Account and to query the balance in that account. It illustrates the use of using stringified object references (IORs) to bootstrap a client process so it can connect to a server.

From this example, you will learn how to:
  • Create a stringified object reference using the org.omg.CORBA.ORB.object_to_string() method
  • Create an object reference from a stringified fromat using the org.omg.CORBA.ORB.string_to_object() method
